# PriceLabs Playwright Automation

## Overview
End-to-end UI automation framework built using Playwright and TypeScript.

## Tech Stack
- Playwright
- TypeScript
- Page Object Model (POM)
- Node.js

## Test Scenarios
- Login
- Billing
- Manage Listings
- Multi Calendar
- Review Price

## Project Structure
```
pages/
tests/
utils/
```

## Run the project

```bash
npm install
npx playwright install
npx playwright test
```

## Features

- Page Object Model
- Reusable locators
- Centralized test data
- Cross-browser support
